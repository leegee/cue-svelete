# `cue-svelte`

Personal project: a first try at Svelte, setting and playing cue markers.


* Input: Video URL
* Click to set cue start
* Highlight start, then click to set cue end 
* Two selections of  cue type:
    * Tension 
    * Action
    * Adventure
    * Sorrow
    * Love
    * Mystery
* Export as SRT and MIDI
